# Инструкция по быстрому запуску проекта

## Метод 1: Команда "ПоехалиИ" через VS Code Tasks

### Как использовать:
1. Нажмите **Cmd+Shift+P** (или F1)
2. Введите `Tasks: Run Task`
3. Выберите задачу **"ПоехалиИ"**

Это автоматически запустит:
- ✅ Backend на http://localhost:3000
- ✅ Frontend на http://localhost:5173

В двух отдельных терминалах в группе "dev"

---

## Метод 2: Bash скрипт

Запустите в терминале:
```bash
./start-dev.sh
```

Это запустит оба сервера в фоновом режиме.

---

## Метод 3: Настроенные терминальные профили

В VS Code теперь доступны специальные терминальные профили:

1. **Backend Terminal** - автоматически открывается в папке `backend/`
2. **Frontend Terminal** - автоматически открывается в папке `frontend/`

### Как создать терминал:
- Нажмите **+** в панели терминала
- Выберите профиль из выпадающего списка

---

## Метод 4: Ручной запуск (для проверки)

### Terminal 1 - Backend:
```bash
cd backend
yarn start:dev
```

### Terminal 2 - Frontend:
```bash
cd frontend
yarn dev
```

---

## Быстрый старт с копилотом

Просто скажите копилоту:
> "Запусти проект" или "ПоехалиИ"

Я запущу оба сервера автоматически!

---

## Остановка серверов

В любом терминале нажмите **Ctrl+C**
